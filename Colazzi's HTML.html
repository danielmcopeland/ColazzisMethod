Colazzi's Method

Colazzi's Method takes data and organizes them based simply on whether the data fall into specific categories.

This is great for qualitative data and finding themes in written responses, but can cause inaccuracies when

dealing with relational data. (especially numerical data) This table uses data from a questionnaire on

appropriate salaries for music therapists to show the differences between using Colazzi's Method and using a

more traditional statistical analysis.

Input salary ranges between $30000 and $90000

(example: 55000-65000)
